Binary Search Algorithm in pseudocode:

Algorithm BinarySearch(arr, target)
Input: arr (sorted array), target (value to find)
Output: Index of target in arr, or -1 if not found

1. Set low ← 0
2. Set high ← length(arr) - 1
3. While low ≤ high do:
    a. Set mid ← (low + high) // 2
    b. If arr[mid] == target:
           Return mid
    c. Else if arr[mid] < target:
           Set low ← mid + 1
    d. Else:
           Set high ← mid - 1
4. Return -1  // Target not found
